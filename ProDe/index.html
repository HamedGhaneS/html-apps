Prosody-on-Demand

AI-Driven Active Earplug for Memory Enhancement through Prosodic Speech Transformation

Dataset Review System

Prosody-on-Demand transforms neutral speech into rhythmic, melodic, prosodic versions to enhance memory encoding. Research shows prosody provides timing cues that make speech easier to encode and recall. This system uses AI to apply content-preserving prosodic patterns (rhythm, rhyme, intonation) to spoken information in real-time, targeting applications in education, safety instructions, and working memory tasks.

The system captures neutral speech, processes it through an active earplug with embedded AI, and delivers prosodic speech to enhance memory encoding. EEG feedback enables real-time optimization of prosodic patterns based on neural responses.
